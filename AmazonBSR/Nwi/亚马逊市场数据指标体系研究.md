摘要

在亚马逊全球市场数据的分析与比较中，不同研究机构、数据平台和咨询公司所引用的数字常常存在显著差异。本附录旨在厘清四个核心指标——1P净销售额、3P GMV、总GMV和电商业务收入——的定义边界、数据来源与方法论差异。

───

1. 指标体系定义

1.1 1P 净销售额（First-Party Net Sales / Revenue）

定义： 亚马逊直接向消费者零售自有库存所获得的全部销售收入。亚马逊从制造商或批发商处买断商品，自主定价并通过自营渠道完成交付。

计算方式：

1P 净销售额 = 商品零售价 × 销售数量

财务含义： 直接体现在10-K年报的"Net Sales"科目中。

数据来源：

| 来源              | 指标名称                                    | 说明                  |
| --------------- | --------------------------------------- | ------------------- |
| Amazon 10-K 年报¹ | North America / International Net Sales | 按地域分拆的官方财务数据        |
| Statista²       | Amazon top markets net sales            | 引用自亚马逊财务披露，整理为结构化图表 |

───

1.2 3P GMV（Third-Party Gross Merchandise Value）

定义： 独立第三方卖家在亚马逊市场上完成交易所产生的商品交易总额。亚马逊不持有库存，仅提供平台与履约服务。

佣金结构示例（美国市场）：

| 角色             | 所得       |
| -------------- | -------- |
| 买家支付           | $100.00  |
| 第三方卖家实收（扣除佣金后） | 约 $85.00 |
| 亚马逊所得（佣金）      | 约 $15.00 |
| 计入3P GMV       | $100.00  |

数据来源：

| 来源                 | 方法论              | 局限性           |
| ------------------ | ---------------- | ------------- |
| Marketplace Pulse³ | 活跃卖家数量 × 平均单卖家收入 | "活跃卖家"定义因平台而异 |
| ECDB⁴              | 基于3P占比估算全口径GMV   | 3P占比系数来自推算    |

───

1.3 总GMV（Total Gross Merchandise Value）

定义： 1P自营 + 3P第三方之和，衡量平台整体交易规模的最宽口径。

计算公式：

总GMV = 1P净销售额 + 3P GMV

同一市场出现多个GMV数字的原因：

| 指标         | 美国市场数值    | 说明                  |
| ---------- | --------- | ------------------- |
| 北美净销售额（1P） | $438B     | 亚马逊自营收入部分           |
| 总GMV（宽口径）  | $830B（全球） | Marketplace Pulse估算 |

注： 4380亿与8300亿美元不是矛盾的——前者是亚马逊留存的收入，后者是平台上所有商品流动的总价值。

数据来源：

| 来源                 | 方法论             | 局限性            |
| ------------------ | --------------- | -------------- |
| Marketplace Pulse³ | 活跃卖家数据 + 平台流量模型 | 非官方披露，估算性质     |
| ECDB⁴              | 网站流量监测 + 电商数据库  | 不同数据源之间的交叉推算   |
| Similarweb⁵        | 流量数据建模为GMV      | 流量≠销售，转换系数存在误差 |

───

1.4 电商业务收入（E-commerce Revenue）

定义： 亚马逊从电商业务中实际获得的所有收入。

收入构成分解：

电商业务收入
  ├── 1P 自营净销售额（Retail）
  ├── 3P 佣金收入（Marketplace Fees）
  ├── 履约费收入（Fulfillment by Amazon / FBA）
  ├── 广告收入（Amazon Advertising）
  └── 订阅服务收入（Amazon Prime）

与GMV的核心区别：

| 指标     | 衡量内容          | 典型数值      |
| ------ | ------------- | --------- |
| 总GMV   | 平台上所有商品流动的总价值 | $830B（全球） |
| 电商业务收入 | 亚马逊实际留存的口袋收入  | $638B（全球） |

2. 数据来源详述

来源一：Amazon 10-K 年报 ¹

机构： Amazon.com (http://amazon.com/), Inc.
性质： 上市公司向SEC提交的年度财务报告
获取地址： https://ir.aboutamazon.com/

报告内容： 按地域拆分的净销售额、按业务线拆分的收入（Online Stores / Third-Party Seller Services / Subscription Services / Advertising / AWS）

优势： ✅ 法定披露，数字最权威
局限： ❌ 不单独披露各国家/地区的3P GMV；GMV非GAAP口径

优势： ✅ 数据结构化、可视化程度高
局限： ❌ 大部分详细数据为付费内容；原始数据来源标注不充分

优势： ✅ 专注亚马逊生态，数据深度高

4. 方法论差异导致的数字分歧

以美国市场为例，同一市场出现多个差异显著的GMV数字：

| 指标            | 数值     | 来源                         | 口径说明       |
| ------------- | ------ | -------------------------- | ---------- |
| 北美净销售额（1P）    | $438B  | Amazon 10-K¹ / Statista²   | 亚马逊自营保留收入  |
| 美国3P GMV      | ~$290B | Marketplace Pulse³         | 第三方卖家总销售额  |
| 美国总GMV（1P+3P） | ~$440B | Marketplace Pulse³ / ECDB⁴ | 宽口径估算      |
| 美国总GMV（最高估算）  | ~$750B | ChannelKey（行业咨询）           | 含所有3P销售全口径 |

结论： 4380亿与7500亿之间的差距（约3100亿）并非亚马逊的亏损，而是第三方卖家保留的部分。混淆"收入"与"GMV"是市场数据报告中最常见的误解来源。
