# 分销商 (Distributor)

> [!INFO] Definition
> **分销商**是买断货权的中间商，从品牌方处购入商品，自主定价销售，承担库存风险，赚取进销差价。

## 核心特征

| 维度 | 说明 |
| :--- | :--- |
| **货权** | 买断（品牌方资金快速回收） |
| **利润来源** | 进销差价（低买高卖） |
| **风险承担** | 高（卖不掉就是自己的库存） |
| **核心任务** | 铺货、周转、配送 |

## 分销商 vs. 代理商

| 维度 | 分销商 | 代理商 |
| :--- | :--- | :--- |
| **货权** | 买断 | 代售/授权 |
| **利润** | 进销差价 | 佣金/返利 |
| **风险** | 高 | 低 |
| **核心任务** | 铺货配送 | 渠道管理 |

## 跨境场景中的分销商

- 线上：Amazon 第三方卖家、eBay 卖家
- 线下：区域批发商、华人超市供货商
- 混合：国家代理（小米模式）

## 痛点

- 多平台分销可能导致价格战（Amazon 跟卖）
- 跨区窜货干扰 BSR 数据
- 品牌方难以获取终端消费者数据
