# BSR (Best Sellers Rank)

> [!INFO] Definition
> **BSR** = Best Sellers Rank，即**亚马逊畅销榜排名**。反映某个 ASIN 在其所属小类目中的实时销售表现。

## 核心要点

- BSR 是 Amazon 平台内反映产品销量的核心指标之一
- 排名数字越小，说明销量越好（#1 > #100 > #10000）
- 通常按类目小节点计算，不同类目竞争激烈程度不同

## BSR 与销量的关系

BSR 可以间接反映销量区间，但需要注意：

- 不同类目客单价不同，BSR 对应的绝对销量差异极大
- BSR 波动可能由季节性、竞争对手变化、评价变化等多因素引起
- **BSR 预测**是解决库存错配的核心手段

## 在 CBEC 中的应用

- 高 BSR 波动 → 需要快速响应补货
- BSR 骤降 → 可能面临库存积压风险
- 结合 BSR 预测，可以优化 FBA 备货策略
