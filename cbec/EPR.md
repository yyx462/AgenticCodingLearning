# EPR (Extended Producer Responsibility)

> [!INFO] Definition
> **EPR** = Extended Producer Responsibility，即**生产者责任延伸**。要求生产者对产品的整个生命周期（尤其废弃物处理）承担责任的环保制度。

## 核心要点

- EPR 是欧洲的环保合规要求，覆盖多个产品类别
- 跨境卖家销往欧洲，需为适用产品类别注册 EPR
- 与 WEEE（废旧电子设备指令）、包装法等密切相关

## 主要产品类别

| 类别 | 说明 |
| :--- | :--- |
| 包装 | 纸、塑料、复合材料等 |
| 电子电气设备 (WEEE) | 家用电器、电子产品 |
| 电池 | 锂电池、纽扣电池等 |
| 纺织品 | 法国 EPR 要求（2023 新增） |
