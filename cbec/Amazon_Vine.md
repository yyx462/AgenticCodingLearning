# Amazon Vine 计划

> [!INFO] Definition
> **Amazon Vine** 是 Amazon 官方提供的评论计划，允许品牌方通过 Vine Voice 社区的真实买家获取合规的产品评论。

## 核心特征

| 维度 | 说明 |
| :--- | :--- |
| **性质** | 官方合规渠道 |
| **评论来源** | Vine Voice 社区的真实买家 |
| **可信度** | 高（买家真实购买并撰写） |

## 与刷单的区别

| 维度 | Amazon Vine | 刷单 |
| :--- | :--- | :--- |
| **合法性** | 官方认可 | 违规（会导致封号） |
| **评论质量** | 高（真实购买体验） | 低（虚假/敷衍） |
| **风险** | 无 | 极高（账号被封） |

## 参与条件

- 在 Amazon Brand Registry 注册的品牌
- 产品需要在 Amazon 平台有销售记录
- 缴纳 Vine 计划费用（按 ASIN 收费）
