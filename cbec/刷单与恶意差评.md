# 刷单与恶意差评攻击 (Brushing & Review Sabotage)

> [!INFO] Definition
> **刷单**是指通过虚假交易制造假评论；**恶意差评攻击**是指竞争对手针对目标 Listing 故意刷入大量 1 星评价并通过点赞使其置顶。

## 核心特征

| 类型 | 行为 | 痛苦点 |
| :--- | :--- | :--- |
| **刷单** | 虚假交易制造假评论 | 触发平台风控封号 |
| **恶意差评** | 刷 1 星 + 点赞置顶 | 转化率暴跌 |

## Amazon Vine 计划

> [!NOTE] 合规获取评论的官方渠道
> Amazon Vine 是品牌方通过官方渠道获取合规评论的计划，评论由 Vine Voice 社区的真实买家撰写。

## 应对策略

| 策略 | 说明 |
| :--- | :--- |
| **官方程序** | 加入 Amazon Vine 计划 |
| **证据链条** | 监控评论增长曲线，发现异常立即报备平台 |
| **品牌注册** | 通过 Brand Registry 获得高级别评论管理权限 |
