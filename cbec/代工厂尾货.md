# 代工厂尾货 / 后门货 (Factory Backdoor)

> [!INFO] Definition
> **代工厂尾货**是指品牌方委托代工厂生产后，工厂私自流出或额外生产的商品。是一种严重的知识产权侵犯行为。

## 核心特征

| 维度 | 说明 |
| :--- | :--- |
| **货的真假** | 接近真品（肉眼难辨） |
| **货从哪里来** | 代工厂私自外流 |
| **典型形式** | 影子工厂多生产、次品尾货倒卖 |
| **品控** | 不受品牌方质量体系管控 |

## 形成机制

### 影子工厂（深夜班次）

品牌方下单 10,000 件 → 工厂利用同样模具、原材料，在"深夜班次"额外生产 1,000 件 → 私自卖给跨境卖家

### 次品/尾货

- 品牌方质检没通过的退货
- 本应销毁的残次品
- 被内部人员偷偷倒卖

## 对品牌方的危害

- **严重侵犯知识产权**
- 品质无法管控，损害品牌形象
- 品牌方难以自查（无法区分正品和尾货）
- 在 [[跟卖_Hijacking]] 中属于"最难投诉"类型
