# 零售套利 (Retail Arbitrage / Online Arbitrage)

> [!INFO] Definition
> **零售套利**是指趁低价买入零售商品，在高价市场（如 Amazon）卖出的行为。个人卖家最常见的起家方式。

## 核心特征

| 维度 | 说明 |
| :--- | :--- |
| **货的真假** | 真品 |
| **货从哪里来** | 零售店促销、电商差价、平台补贴商品 |
| **典型来源** | 沃尔玛/Target 清仓、拼多多/Temu 低价货源 |
| **规模** | 通常量小，但足以扰乱价格 |

## 在 CBEC 中的应用

- **个人卖家起家路径：** 趁节假日促销大量扫货，在 Amazon 跟卖热门 ASIN
- **对品牌方的影响：** 瞬间拉低价格，扰乱 [[BSR]] 稳定性
- **投诉难点：** 由于是正品，举报难以成功
