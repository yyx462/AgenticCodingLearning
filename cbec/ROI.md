# ROI (Return on Investment)

> [!INFO] Definition
> **ROI** = Return on Investment，即**投资回报率**。衡量投入产出效率的核心财务指标。

## 核心公式

```
ROI = (收入 - 成本) / 成本 × 100%
```

## 在 CBEC 中的应用

### 全链路成本构成（通常）

| 成本项 | 占比 |
| :--- | :--- |
| 商品成本 | 30-40% |
| 物流费 | 20-30% |
| 平台佣金 | 8-15% |
| 广告费 | 10-20% |
| 退货/损耗 | 3-8% |
| **净利** | **个位数** |
