# 代理商 (Agent / Representative)

> [!INFO] Definition
> **代理商**是受品牌方委托，代行品牌职责（渠道管理、营销、售后等）的中间商，通常不买断货权，以佣金或返利为主要收入来源。

## 核心特征

| 维度 | 说明 |
| :--- | :--- |
| **货权** | 代售/授权（货权在品牌方） |
| **利润来源** | 佣金 (Commission) 或品牌返利 |
| **风险承担** | 低（不承担库存风险） |
| **核心任务** | 品牌维护、渠道管理、售后服务 |

## 代理商 vs. 分销商

| 维度 | 代理商 | 分销商 |
| :--- | :--- | :--- |
| **货权** | 代售/授权 | 买断 |
| **利润** | 佣金/返利 | 进销差价 |
| **风险** | 低 | 高 |
| **核心任务** | 渠道管理 | 铺货配送 |

## 跨境场景中的代理商

- 线下：区域总代（如海尔在欧美的家电代理）
- 专业渠道：Best Buy、Lowe's 等大型连锁谈判代理
- 垂直场景：韶音在跑步装备店的代理

## 痛点

- 售后承诺难以兑现，品牌承担骂名
- 渠道冲突（线上 vs. 线下价格矛盾）
- 代理商离开可能导致渠道断层
