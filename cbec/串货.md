# 串货 / 水货 (Unauthorized Distribution / Gray Market)

> [!INFO] Definition
> **串货**是指商品未经品牌方授权，跨区域流通销售的行为。是品牌方地区价格歧视策略下的灰色产物。

## 核心特征

| 维度 | 说明 |
| :--- | :--- |
| **货的真假** | 绝对正品 |
| **货从哪里来** | 其他地区的非授权分销商 |
| **典型场景** | 中国区代理将货发往美国 Amazon 销售 |
| **价格** | 低于品牌方海外旗舰店的零售价 |

## 形成机制

品牌方通常对不同地区采用**价格歧视策略**：

- 中国区代理拿货价：100 元
- 美国区代理拿货价：150 元
- 中国区分销商将货发往美国，以 130 元跟卖美区旗舰店

## 对品牌方的危害

- 扰乱全球定价逻辑
- 引发各区域代理商之间的内斗
- 利润被中间商蚕食
- **无法通过平台投诉解决**（因为是正品）
