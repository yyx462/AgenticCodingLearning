# 恶意点击 (Click Fraud)

> [!INFO] Definition
> **恶意点击**是指竞争对手利用 Bot（机器人）或雇佣"点击公司"，对目标广告进行大量无效点击的行为。

## 核心特征

| 维度 | 说明 |
| :--- | :--- |
| **目的** | 消耗对手广告预算，降低转化率 |
| **手段** | Bot 点击、点击公司真人点击 |
| **影响** | 广告费烧光、转化率暴跌、算法降权 |

## 对 [[BSR]] 的间接影响

- 无转化点击 → 平台算法判定产品"转化率低"
- 自然搜索排名下降
- [[PPC广告]] 权重降低

## 应对策略

| 策略 | 说明 |
| :--- | :--- |
| **技术手段** | 使用广告防欺诈工具，识别并屏蔽可疑 IP |
| **运营策略** | 设置分时段竞价，避开竞争对手活跃时段 |
| **平台申诉** | 向平台提交异常流量证据，申请广告费返还 |
