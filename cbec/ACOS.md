# ACOS (Advertising Cost of Sales)

> [!INFO] Definition
> **ACOS** = Advertising Cost of Sales，即**广告销售成本比**。衡量广告投放效率的关键指标。

## 核心公式

```
ACOS = 广告费用 / 广告带来的销售额 × 100%
```

## 核心要点

- ACOS 越低，说明广告效率越高
- 与 [[ROAS]] 的关系：`ROAS = 1 / ACOS`

| ACOS 范围 | 含义 |
| :--- | :--- |
| 0-15% | 效率极高（竞争不激烈的利基市场） |
| 15-30% | 健康水平（大多数品类） |
| 30-50% | 竞争激烈，流量成本较高 |
| >50% | 广告亏损，需优化策略 |
