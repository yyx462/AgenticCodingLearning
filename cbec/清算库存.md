# 清算库存 (Liquidation)

> [!INFO] Definition
> **清算库存**是指企业破产清算或线下清仓时，库存被以极低价格打包出售给职业清算人，再由清算人分批给跟卖者在 Amazon 上套现。

## 核心特征

| 维度 | 说明 |
| :--- | :--- |
| **货的真假** | 真品 |
| **货从哪里来** | 破产清算、线下连锁店倒闭 |
| **价格** | 极低（通常为原价的一折甚至更低） |
| **特点** | 不受控的低价抛售 |

## 形成机制

1. 线下连锁店倒闭或品牌方大代理商破产
2. 库存被以极低价格打包出售给"职业清算人"
3. 清算人分批给跟卖者去 Amazon 上套现
4. 跟卖者以低于市场价的姿态快速出货

## 对 [[BSR]] 的影响

- 短期冲击 [[BSR]] 排名（大量低价出单）
- 价格低于品牌方官限价
- 可能被误判为市场真实需求增长
