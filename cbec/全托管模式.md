# 全托管模式 (Full Managed Service)

> [!INFO] Definition
> **全托管模式**是近两年中国跨境电商平台（Temu、AliExpress、TikTok Shop）推出的新型卖家服务模式，平台承担运营、履约、售后，卖家只需供货。

## 核心要点

- 卖家 = 供货商，平台 = 运营方
- 典型特征：卖家无定价权、货物发至平台仓库
- 核心优势：**门槛极低**，有货就能卖

## 全托管 vs. POP 对比

| 维度 | 全托管模式 | POP 模式 (3P) |
| :--- | :--- | :--- |
| **运营主体** | 平台 | 卖家 |
| **定价权** | 平台统一定价 | 卖家自主定价 |
| **资金占用** | 低（无需提前备货） | 高（需要备货/广告投入） |
| **品牌积累** | 弱 | 强 |
| **利润空间** | 极度压缩 | 相对较高 |

> [!WARNING] 核心风险
> 全托管模式利润空间被极度压低，**适合有工厂资源但不擅长流量运营的传统制造商**。
