# IP (Intellectual Property)

> [!INFO] Definition
> **IP** = Intellectual Property，即**知识产权**。涵盖商标、专利、著作权、外观设计等法律权利。

## 核心要点

- 跨境电商中，IP 主要涉及：商标侵权、专利侵权、著作权（图片/文案）侵权
- 平台（尤其 Amazon）对 IP 侵权采取"举报即下架"政策
- 封号风险：轻则产品下架，重则店铺被封、资金冻结

## 主要风险类型

| 类型 | 说明 | 风险等级 |
| :--- | :--- | :--- |
| **商标侵权** | 使用他人已注册商标 | 高 |
| **专利侵权** | 发明/外观专利未经授权 | 极高 |
| **著作权侵权** | 盗用图片、文案、产品描述 | 中 |
| **trade dress** | 产品外观、包装混淆 | 高 |

## IP 保护策略

1. **事前排查：** 销售前进行商标/专利检索
2. **自有品牌：** 注册自有商标（如 Amazon 品牌备案）
3. **公模 vs. 私模：** 公模产品易被跟卖，私模更安全
4. **产品认证：** CE、FCC、FDA 等认证可间接规避专利风险
