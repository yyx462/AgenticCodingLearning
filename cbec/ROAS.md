# ROAS (Return on Ad Spend)

> [!INFO] Definition
> **ROAS** = Return on Ad Spend，即**广告支出回报率**。衡量每单位广告支出带来的收入。

## 核心公式

```
ROAS = 广告带来的销售额 / 广告费用
```

## 核心要点

- ROAS 越高，广告效率越高
- ROAS = 10 意味着花 $1 广告带来 $10 销售额
- 与 [[ACOS]] 的关系：`ROAS = 1 / ACOS`

| ROAS 范围 | 含义 |
| :--- | :--- |
| <3 | 广告效率低，难以盈利 |
| 3-5 | 盈亏平衡附近（取决于毛利） |
| 5-10 | 效率良好 |
| >10 | 广告效率极高 |
