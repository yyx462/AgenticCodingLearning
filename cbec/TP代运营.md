# TP (Third-party Partner / 代运营)

> [!INFO] Definition
> **TP** = Third-party Partner，即**代运营服务商**。在跨境电商语境中，通常指帮助品牌方运营电商店铺的第三方服务公司。

## 核心要点

- TP = 代运营 / 运营外包服务商
- 典型服务内容：店铺日常运营、广告投放、客服、listing 优化
- 收费模式：服务费 + 销售额提点（佣金）

## TP vs. 自运营

| 维度 | TP 代运营 | 自运营 |
| :--- | :--- | :--- |
| **启动速度** | 快（团队现成） | 慢（需组建团队） |
| **成本** | 服务费 + 提点 | 固定人力成本 |
| **专业度** | 高（经验积累） | 取决于团队水平 |
| **数据控制** | 弱（数据在 TP 方） | 强（自有） |
| **依赖风险** | 高（TP 离开 = 店铺停摆） | 低 |
