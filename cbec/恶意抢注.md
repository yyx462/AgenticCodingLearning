# 恶意抢注 / IP Trolling

> [!INFO] Definition
> **恶意抢注**是指对方抢先注册你的产品外观设计、品牌名称或常用描述性词汇，然后利用平台规则发起侵权投诉，导致你的爆款被瞬间下架。

## 核心特征

| 维度 | 说明 |
| :--- | :--- |
| **目的** | 打击竞争对手，获取不当利益 |
| **手法** | 抢注商标/外观专利 → 发起侵权投诉 → Listing 被下架 |
| **后果** | 申诉周期数周至数月，销量和排名损失毁灭性 |

## 预防策略

| 策略 | 实施时机 |
| :--- | :--- |
| **商标注册** | 出海前完成目标市场商标注册 |
| **外观专利** | 出海前完成外观专利注册 |
| **知识产权库** | 建立自有知识产权库并在平台备案 |

## 应对策略

| 策略 | 说明 |
| :--- | :--- |
| **反向投诉** | 准备律师函，快速发起 Counter-Notice |
| **平台申诉** | 通过官方渠道提交申诉材料 |
| **法律手段** | 在目标市场发起知识产权诉讼 |
