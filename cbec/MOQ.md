# MOQ (Minimum Order Quantity)

> [!INFO] Definition
> **MOQ** = Minimum Order Quantity，即**最小起订量**。供应商愿意接受的单次最低采购数量。

## 核心要点

- 供应商设定 MOQ 是为了覆盖生产成本和换线成本
- MOQ 越高，单件成本越低，但资金压力和库存风险越大
- 跨境电商卖家的典型困境：**MOQ 门槛 vs. 测品需求**

## Trade-off

| MOQ 高 | MOQ 低 |
| :--- | :--- |
| 单件成本低 | 单件成本高 |
| 适合爆款/稳定款 | 适合测品/定制款 |
| 资金压力大 | 资金压力小 |
