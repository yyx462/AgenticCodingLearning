# 透明计划 (Transparency)

> [!INFO] Definition
> **Transparency（透明计划）**是 Amazon 提供的产品溯源服务，通过在每件商品上贴唯一的加密二维码，确保只有正品才能进入 Amazon 仓库。

## 核心特征

| 维度 | 说明 |
| :--- | :--- |
| **机制** | 每件货贴唯一加密二维码 |
| **验证** | 入 FBA 仓库时扫码验证 |
| **效果** | 无码跟卖者无法入库 |

## 防伪能力

| 欺诈类型 | Transparency 防护 |
| :--- | :--- |
| [[跟卖_Hijacking]] | 跟卖者无码无法入库 |
| [[假货]] | 假货无正确编码被拦截 |
| [[退货欺诈]] | 退货时核对编码，识别掉包 |

## 品牌方收益

- 防止假货和跟卖
- 提升消费者信任（扫码验真）
- 减少退货欺诈
