# 恶意退货与换货欺诈 (Return Fraud / Parts Swapping)

> [!INFO] Definition
> **恶意退货**是指消费者或竞争对手伪装的消费者购买正品后，以"产品损坏"为由退回坏掉的旧机器、石头，或拆走核心零件。

## 核心特征

| 维度 | 说明 |
| :--- | :--- |
| **形式** | 退回损坏品、空包、石头、拆走核心零件 |
| **痛苦点** | 品牌方损失货值 + 退货运费 + 平台处理费 |
| **规模化** | 有组织的欺诈行为可造成重大损失 |

## 典型场景

- 电子产品：拆走驱动单元、电池等核心零件，退回外壳
- 服装：穿过后退回（外貌无损）
- 配件：退回仿品而保留正品

## 应对策略

| 策略 | 说明 |
| :--- | :--- |
| **透明计划 (Transparency)** | 每件货贴唯一加密二维码，退货时核对编码 |
| **库存强制检查** | 要求平台对 FBA 退回库存进行强制检查，不简单重新上架 |
| **黑名单系统** | 在独立站渠道建立高风险客户识别系统 |
