# VAT (Value Added Tax)

> [!INFO] Definition
> **VAT** = Value Added Tax，即**增值税**。欧洲各国对商品和服务征收的间接税，跨境电商卖家必须注册和申报。

## 核心要点

- 欧洲各国（英、德、法、意、西等）均有 VAT 要求
- 跨境卖家需要在销售国注册 VAT 并定期申报
- Amazon 等平台已开始代扣代缴（VAT OSS 机制）

## 主要市场的 VAT 税率

| 国家 | 标准税率 |
| :--- | :--- |
| 德国 | 19% |
| 法国 | 20% |
| 英国 | 20% |
| 意大利 | 22% |
| 西班牙 | 21% |

## 合规要求

1. **注册义务：** 远程销售额超过阈值（通常 €10,000）需在销售国注册
2. **申报周期：** 季度申报或月度申报
3. **税务计算：** 售价含 VAT，消费者支付
