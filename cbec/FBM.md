# FBM (Fulfillment by Merchant)

> [!INFO] Definition
> **FBM** = Fulfillment by Merchant，即**自发货**。卖家自行负责仓储、包装、配送及售后，Amazon 仅提供平台不参与履约。

## 核心要点

- FBM = Merchant Fulfillment，与 FBA 相对
- 常见形式：小包直邮（ePacket）、海外仓一件代发
- 适合：测品阶段、定制化产品、高客单价商品

## FBM 的 Trade-off

| 优点 | 缺点 |
| :--- | :--- |
| 无需囤货，资金压力小 | 时效慢（7-15天），影响转化率 |
| 灵活多变，适合测品 | 无法获得 Prime 标记 |
| 无仓储费用 | 物流成本受航空运价波动影响大 |
| 可掌控退换货流程 | 权重低于 FBA |
