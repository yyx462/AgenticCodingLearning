# FBA (Fulfillment by Amazon)

> [!INFO] Definition
> **FBA** = Fulfillment by Amazon，即**亚马逊物流**。卖家将商品批量发送至 Amazon 运营中心，由 Amazon 负责存储、拣货、包装、配送及售后。

## 核心要点

- FBA 是 Amazon 平台的核心履约模式
- Amazon 负责尾程配送（up to last mile），提供 Prime 资格
- 费用包括仓储费 + 配送费（按体积/重量计算）

## FBA 的 Trade-off

| 优点 | 缺点 |
| :--- | :--- |
| Prime 会员配送快，转化率高 | 仓储和配送费用较高 |
| 权重高，搜索排名有利 | 需要提前囤货，资金压力大 |
| 客服和退货由 Amazon 处理 | 库存积压风险，旺季附加费 |
| 可参与 Prime Day 等活动 | 标准化程度要求高 |

## FBA vs. FBM

| 模式 | 含义 | 特点 |
| :--- | :--- | :--- |
| **FBA** | Amazon 负责履约 | 速度快，Prime 资格，但成本高 |
| **FBM** | 卖家自发货 | 灵活，成本低，但速度慢 |
